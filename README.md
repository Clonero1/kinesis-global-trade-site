# Сайт Kinesis Global Trade

Проект B2B/B2C-сайта для ООО «Кинезис Глобал Трейд».

Цель проекта — создать сайт-каталог для продажи запчастей для карьерной, горнодобывающей, строительной и спецтехники с возможностью дальнейшего развития до маркетплейса.

## Основные задачи сайта

1. Представить компанию и направление продаж запчастей.
2. Разместить каталог товаров с ценами, остатками, артикулами и характеристиками.
3. Реализовать быстрый поиск по артикулу.
4. Реализовать массовый поиск товаров списком артикулов.
5. Организовать загрузку товаров, цен и остатков из 1С, Excel или XML.
6. Дать возможность покупателю отправлять заявку или оформлять заказ.
7. Реализовать личный кабинет покупателя для физических и юридических лиц.
8. Реализовать автоматическое формирование счета/договора для юридических лиц.
9. Реализовать расчет доставки через транспортные компании, в первую очередь СДЭК и Деловые Линии.
10. Подготовить архитектуру для будущего масштабирования до маркетплейса с личным кабинетом продавца.

## Ключевой принцип по товарам

Артикул является главным идентификатором товара. Нельзя считать товары взаимозаменяемыми только по похожему названию. Поиск, импорт, сопоставление и обновление карточек должны опираться прежде всего на каталожные номера и артикулы.
