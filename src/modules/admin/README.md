# Admin module

## Назначение

Модуль админки отвечает за будущий технический и операционный контроль данных сайта.

## Что модуль должен делать в будущем

- показывать товары;
- показывать импорты и синхронизации;
- показывать заявки;
- показывать статусы отправки в 1С;
- показывать логи ошибок интеграции;
- давать ручной повтор отправки заявки, если это безопасно.

## Что модуль пока не делает

- не создает административный интерфейс;
- не реализует авторизацию;
- не редактирует товарную истину вместо 1С;
- не управляет документами как бухгалтерская система.
