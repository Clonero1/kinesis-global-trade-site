# Bulk search module

## Назначение

Модуль массового поиска отвечает за будущую обработку списков артикулов.

## Что модуль должен делать в будущем

- принимать список артикулов;
- нормализовать каждую строку отдельно;
- искать товары по `ProductIdentifier.normalizedValue`;
- возвращать результат по каждой исходной строке;
- показывать найденные и ненайденные позиции;
- показывать цену, остаток, склад и состояние;
- позволять добавить найденные позиции в заявку.

## Что модуль пока не делает

- не создает UI массового поиска;
- не создает API;
- не импортирует файлы;
- не отбрасывает ненайденные строки;
- не объединяет товары по похожему названию.
