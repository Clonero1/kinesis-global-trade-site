# Delivery module

## Назначение

Модуль доставки отвечает за будущий расчет и хранение выбранного способа доставки по заявке.

## Что модуль должен делать в будущем

- хранить доставку в `DeliveryInfo`;
- поддерживать СДЭК;
- поддерживать Деловые Линии;
- хранить адрес, город, пункт выдачи или терминал;
- передавать выбранную доставку в 1С вместе с заявкой.

## Что модуль пока не делает

- не подключает СДЭК;
- не подключает Деловые Линии;
- не рассчитывает тарифы;
- не оформляет доставку;
- не блокирует сохранение заявки при недоступности перевозчика.
