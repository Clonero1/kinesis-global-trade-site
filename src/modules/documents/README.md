# Documents module

## Назначение

Модуль документов отвечает за будущую работу с документами, которые сформированы в 1С и доступны сайту как ссылка или файл.

## Что модуль должен делать в будущем

- хранить записи `Document`;
- связывать документы с заявками;
- показывать номер, тип, статус и дату документа;
- хранить ссылку или ключ файла;
- обеспечивать доступ только владельцу заявки или администратору.

## Что модуль пока не делает

- не формирует счета;
- не формирует договоры;
- не формирует УПД или накладные;
- не создает печатные формы;
- не подключается к 1С.

## Важные правила

Официальные документы формирует 1С. Сайт только получает и отображает документы или ссылки на них.
