# Integrations module

## Назначение

Модуль интеграций отвечает за будущий обмен сайта с внешними системами.

## Что модуль должен делать в будущем

- создавать и обрабатывать `IntegrationTask`;
- писать события в `IntegrationLog`;
- поддерживать повторные попытки;
- использовать идемпотентные ключи;
- изолировать внешние системы от бизнес-модулей сайта.

## Что модуль пока не делает

- не подключает реальную 1С;
- не подключает Битрикс24;
- не подключает СДЭК;
- не подключает Деловые Линии;
- не формирует XML для Авито;
- не реализует очередь.
