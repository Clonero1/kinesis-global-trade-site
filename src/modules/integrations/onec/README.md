# 1C integration module

## Назначение

Модуль интеграции с 1С отвечает за будущий обмен товарами, ценами, остатками, складами, заявками, статусами заказов и документами.

1С является главным источником товаров, цен, остатков, складов и документов. Сайт не заменяет 1С и не становится главным источником складской, бухгалтерской или документальной истины.

## Что модуль должен делать в будущем

- получать товары из 1С;
- получать цены из 1С;
- получать остатки и склады из 1С;
- отправлять заявки в 1С;
- получать статусы заказов;
- получать документы или ссылки на документы;
- использовать очередь повторной отправки;
- использовать идемпотентный ключ при отправке заявки.

## Что модуль пока не делает

- не подключается к реальной 1С;
- не отправляет HTTP-запросы;
- не создает заказы в 1С;
- не получает документы;
- не хранит реальные учетные данные;
- не формирует официальные документы на сайте.

## Важные правила

Заявки сначала сохраняются в базе сайта. Только после этого отправка в 1С должна идти через очередь и идемпотентный ключ.

Если 1С недоступна, заявка остается на сайте и должна быть отправлена повторно позже.
