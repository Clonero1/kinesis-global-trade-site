# Search module

## Назначение

Модуль поиска отвечает за будущий точный поиск товара по артикулу.

## Что модуль должен делать в будущем

- нормализовать пользовательский ввод;
- искать по `ProductIdentifier.normalizedValue`;
- возвращать товар, состояние, цену, остаток и склад;
- явно показывать несколько найденных вариантов;
- использовать `src/lib/article-normalization.ts`.

## Что модуль пока не делает

- не создает страницу поиска;
- не реализует API;
- не ищет по названию товара как по основному признаку;
- не определяет аналоги;
- не скрывает неоднозначные совпадения.

## Важные правила

Похожее название не является аналогом. Если найдено несколько вариантов по артикулу или идентификатору, их нельзя скрывать и нельзя автоматически выбирать один вариант без явного правила.
